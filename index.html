SRT to VTT Converter (for Windows)

A simple application for quickly converting subtitle files from SubRib (.srt) format to WebVTT (.vtt) format. If you don't need a GUI or you need a version for OS X, try the command line version.

Download

Download the appropriate version and run it, no installation necessary!

  • 32-bit version
  • 64-bit version

Features

  • Select multiple .srt files to be converted all at once
  • Shift the time that the subtitles appear on screen using the timing offset tool
